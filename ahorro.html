Problema Real: Proyección Financiera con Fibonacci

01

1. Contexto del Problema

Muchas personas buscan planes de ahorro progresivos. Este modelo utiliza la secuencia de Fibonacci multiplicada por una base de 100 dólares/pesos para calcular depósitos mensuales que crecen de forma sostenible.

02

2. Explicación del Algoritmo

Se genera la serie sumando los dos meses anteriores ($F_n = F_{n-1} + F_{n-2}$). El resultado puro (0, 1, 1, 2, 3...) se multiplica por 100 para generar montos financieros reales.

ahorro_fibo.js
function calcularProyeccion(meses) {
    let a = 0, b = 1, c;
    let acumulado = 0;
    
    for (let i = 1; i <= meses; i++) {
        let deposito = a * 100; // Escalado financiero
        acumulado += deposito;
        
        // Sucesión de Fibonacci F(n) = F(n-1) + F(n-2)
        c = a + b;
        a = b;
        b = c;
    }
    return acumulado;
}
