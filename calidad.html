Control de Calidad con Números Primos

LED • 01

1. Contexto del Problema

En las fábricas, no se puede revisar cada pieza producida. Para evitar sesgos, los ingenieros seleccionan las piezas cuyos números de serie o lote coinciden con números primos para aplicarles pruebas destructivas de calidad.

LED • 02

2. Explicación del Algoritmo

El sistema toma el número de pieza y cuenta cuántos divisores exactos tiene. Si los divisores son exactamente 2 (el 1 y sí mismo), marca la pieza como "Crítica para revisión".

inspector_primos.js
function esPiezaCritica(numeroSerie) {
    if (numeroSerie <= 1) return false;
    let divisores = 0;
    
    // Algoritmo de primalidad por fuerza bruta
    for (let i = 1; i <= numeroSerie; i++) {
        if (numeroSerie % i === 0) {
            divisores++; // Contador de residuos exactos
        }
    }
    // Si solo tiene 2 divisores (1 y sí mismo) es primo
    return divisores === 2;
}
