Generador de Tokens

Criptografía con Fibonacci + Primos

1. Contexto del Problema

Los sistemas bancarios y de servidores necesitan generar contraseñas de un solo uso (Tokens). Este sistema mezcla el crecimiento exponencial de Fibonacci y lo enmascara usando la validación de primalidad.

LED • 02

2. Explicación del Algoritmo

El algoritmo recibe una "semilla" del usuario. Calcula la posición de Fibonacci para esa semilla, genera un código alfanumérico, y finalmente dictamina si el Token es de Nivel Alto (si su base es prima) o Estándar.

auth_token.js
function generarToken(semillaUsuario) {
    // Extraemos el factor de la serie Fibonacci
    let factorBase = obtenerFibonacci(semillaUsuario);
    let tokenHash = `AUTH-${factorBase * 10}-X9`;
    
    // Verificación cruzada: ¿La base es un número primo?
    let esSeguro = esPrimo(factorBase);
    
    return {
        hash: tokenHash,
        nivelSeguridad: esSeguro ? "ALTO (Primo)" : "ESTÁNDAR"
    };
}
